# My Rails Blog App

> The official Rails blog tutorial.

## Built With

- Major languages: Ruby
- Framework: [Rails](https://rubyonrails.org/)

## Usage

Run the following commands:</br>
`git clone https://github.com/Hombre2014/Blog.git`</br>
`cd Blog`</br>
`bundle install`</br>
`rails s`</br>
Open a browser and go to: `https://localhost:3000` to view the app.
